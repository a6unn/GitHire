# Enhanced Components

This directory contains enhanced versions of existing components with improved UI/UX.

## Components

- **CandidateCardEnhanced** - Beautiful candidate result card with score visualizations
- **PipelineProgressEnhanced** - Modern pipeline progress with glassmorphism and animations
- **ProjectCardEnhanced** - Enhanced project list item card

## Migration Strategy

These components will gradually replace the basic versions in the main components directory.

## Usage

```tsx
import { CandidateCardEnhanced } from '@/components/enhanced/CandidateCardEnhanced';
```
